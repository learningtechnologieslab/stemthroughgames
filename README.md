# STEM Through Games

The University of Pittsburgh's **STEM Through Games Enrichment Program** aims to introduce computational thinking, computer programming, and STEM-related fields to middle school students. The program employs a workshop format centered on digital game design as a gateway to STEM skills, using platforms & programming languages like Python, Scratch, Minecraft, and Godot.

**Project Goals and Objectives:**
* **Exposure**: Introduce underrepresented students to computing fields by engaging them in game design.
* **Skill Development**: Foster data literacy, computational thinking, and foundational programming skills.
* **Curriculum Creation**: Develop and open-source a game-based STEM curriculum.
* **Program Evaluation**: Measure the curriculum’s effectiveness in enhancing STEM interest and skills among students.
